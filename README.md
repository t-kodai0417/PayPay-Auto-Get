# PayPay-Auto-Get
PayPay自動受け取りのソースコード＆サンプルコード

# 用意する物
- PayPayのアカウント
- Pythonを動かせるPCなど

# 使い方
- config.jsonに電話番号とかいろいろ入力。
- client_uuidがわからなかったら`get_client_uuid.py`を実行して取得。
- サンプルコードはexamplesの中に入ってるので参考程度に。
